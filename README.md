# NilBridge2

## 什么是NilBridge？

 - NilBridge是与基岩版服务端Bedrock Dedicated Server与QQ通讯的基础框架，拥有海量十几个API接口。

 - 通过使用js等语言编写插件，开发者可以方便地对NilBridge进行功能拓展与特性定制，容易上手，并且具有极高的灵活性.

 - 对于用户来说，NilBridge可以做到简易上手，开箱即用，对于新手用户非常友好。


## NilBridge的优点

   -  基于Node.js，全平台兼容
   - 全开源代码
   - 性能优化极佳
   - 文档完善
   - 支持插件拓展
   - 开箱即用
